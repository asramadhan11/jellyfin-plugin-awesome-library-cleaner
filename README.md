## üõ†Ô∏è Using the Plugin

Once the plugin is installed, follow these steps to clean your library:

1. Go to the Jellyfin main menu.
2. Click on "Library" and select the media section you want to clean (e.g., Movies or TV Shows).
3. Locate the "Awesome Library Cleaner" option in the menu.
4. Set criteria for cleaning, such as:
   - Remove items not watched in the last 6 months.
   - Delete duplicates.
5. Click "Clean Library" to start the process.
6. Review the items to be removed and confirm the action.
